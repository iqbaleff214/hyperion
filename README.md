# Hyperion

**Hyperion** is a desktop application designed to **secure source code** using advanced **obfuscation techniques**. It supports multiple programming languages, including:  
- **JavaScript**
- **PHP**
- **And other interpreted formats**  

With **Hyperion**, you can **scramble your code** to make it difficult to read or modify while maintaining its original functionality.
